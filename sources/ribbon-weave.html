Ribbon Weave

Short woven ribbons alternate direction behind the pointer.

copy source

Use this effect

<section class="cursor-demo" data-cursor-effect="ribbon-weave">
  <div class="fx-layer" aria-hidden="true"></div>
</section>
.cursor-demo {
  position: relative;
  min-height: 320px;
  overflow: hidden;
  border-radius: 24px;
}

.fx-layer {
  position: absolute;
  inset: 0;
  pointer-events: none;
}
const effect = COLD_EFFECTS.find((item) => item.key === "ribbon-weave");
const target = document.querySelector("[data-cursor-effect='ribbon-weave']");
const layer = target.querySelector(".fx-layer");

target.addEventListener("pointermove", (event) => {
  const rect = layer.getBoundingClientRect();
  COLD_FX.spawn(effect, layer, event.clientX - rect.left, event.clientY - rect.top);
});
Theme aware effect: dark and light palettes are included in assets/data.js.
